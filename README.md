## üìñ Overview

FireAndForget is a lightweight Kotlin Multiplatform library designed to execute code only once at first access. It is ideal for onboarding flows, feature announcements, and other one-time operations. This library works seamlessly across Android, iOS, Desktop, and Web platforms, making it an excellent choice for developers and product managers.

## üåü Features

- **Multiplatform Support:** Use FireAndForget on Android, iOS, Desktop, and Web with a single library.
- **Easy Setup:** Get up and running quickly with straightforward instructions.
- **One-Time Execution:** Ensure that specific actions only occur once, enhancing user experience during onboarding and feature rollouts.
- **State Management:** Maintain the state of user interactions across sessions effortlessly.
- **Feature Flags:** Activate features based on user actions or conditions seamlessly.
